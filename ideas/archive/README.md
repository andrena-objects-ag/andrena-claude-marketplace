# Ideas Archive

This directory contains ideas that have been processed by the `/implement-idea` command.

## Structure

- **implemented/**: Ideas that have been implemented as plugin features
  - Status: `IMPLEMENTED` or `ENHANCED_EXISTING`
  - Contains reference to where the feature was implemented
  - Includes implementation date

- **duplicates/**: Ideas that were found to duplicate existing features
  - Status: `DUPLICATE`
  - Contains reference to existing feature
  - Includes date when duplicate was detected

## Why Archive?

Archiving keeps the main `ideas/` folder clean and focused on active, unimplemented ideas while preserving the history of what was considered and implemented.

## Retrieving Archived Ideas

All archived ideas maintain their frontmatter with:
- Original description
- Status (IMPLEMENTED, DUPLICATE, ENHANCED_EXISTING)
- Reference to implementation or existing feature
- Date processed

You can search archived ideas to:
- See what has already been implemented
- Avoid duplicating work
- Find inspiration for enhancements
- Track the evolution of the marketplace
