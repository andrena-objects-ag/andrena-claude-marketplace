# Gemini Web Search Agent

> Specialized web search and information retrieval using the Gemini CLI

## Overview

The Gemini Web Search Agent is a specialized agent for performing web searches and retrieving current information using the Gemini CLI. Perfect for research, documentation lookup, and finding up-to-date information.

## Installation

```bash
/plugin marketplace add p-wegner/claude-marketplace
/plugin install gemini-web-search-agent@claude-marketplace
```

## Usage

The agent is automatically available after installation and can be used for any web search tasks requiring current information.

### Example Usage:
```
Search for the latest Claude Code documentation updates
Find current best practices for React development
Research recent developments in AI technology
```

## Features

- **Specialized Web Search**: Optimized search queries using Gemini CLI
- **Current Information**: Real-time information retrieval
- **Documentation Search**: Find relevant documentation quickly
- **Research Support**: Track latest developments in any field
- **Explicit Instructions**: Clear patterns for web search tasks

## Requirements

- Gemini CLI installed and configured
- Internet access for web searches
- Claude Code with plugin support

## Configuration

Ensure the Gemini CLI is properly configured with your API keys and settings before use.
